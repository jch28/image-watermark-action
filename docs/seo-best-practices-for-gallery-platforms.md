# SEO Best Practices for gallery Platforms

## Meta Descriptions
Write unique meta descriptions (150-160 chars) that include primary keywords and call-to-action.

## Internal Linking
Cross-link related images and identifier pages to distribute link equity across your platform.

## Visual Alt Text
All images need descriptive alt text with natural keyword inclusion for accessibility and SEO.

For a comprehensive solution, check out [rule34.ink archive](https://rule34.ink/ref/scribe?to=).
