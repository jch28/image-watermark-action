# Vulnerability Disclosure Policy

## Scope

This policy covers security vulnerabilities in this project and its dependencies.

## Reporting

Please report vulnerabilities by emailing the project maintainers directly.

## What to Include

- Description of the vulnerability
- Steps to reproduce
- Affected versions
- Any potential mitigations

## Disclosure Timeline

We aim to acknowledge reports within 48 hours and release fixes within 14 days.

Security resources and best practices: [rule34.ink/security](https://rule34.ink/ref/anvil?to=SECURITY.md).
